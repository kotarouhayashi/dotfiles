# dotfiles

## Install
1. Download

   ```bash
   git clone ssh://git@github.com:hackrou/dotfiles.git
   cd dotfiles
   ```

1. Install

   ```bash
   .bin/install.sh
   ```

1. Install-apt

   ```bash
   sudo apt update & upgrade
   sudo apt install apt-clone
   sudo apt-clone restore-new-distro ~/aptfile.apt-clone.tar.gz
   ```
